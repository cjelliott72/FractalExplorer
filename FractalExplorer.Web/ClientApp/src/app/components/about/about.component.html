About Fractal Explorer

Dependency Information

This application was written using Visual Studio 2019 Community Edition version 16.4.0

The web client is built using Angular 8 (build 8.3.19) and Angular Material UI which requires Node.js version 12.13 to be installed. If you have the above version of Visual Studio and Node.js installed, all you need to do is load, build and run the application. The first time you build the application, Nuget will automatically install the required packages and this can take a few minutes to complete.

Project Architecture

The project uses the ASP.Net Core 3.0 Web Application template for Angular 8. This allows the Angular client and the server Web API to reside in the same project which is ideal for prototype applications like this.

The FractalRenderer library provides the functionality to calculate and generate the images. This is used by the FractalExplorer.Web API to expose the library methods over the web.
